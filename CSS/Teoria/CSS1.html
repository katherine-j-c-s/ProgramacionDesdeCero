Cómo incluir CSS en un HTML

En el mismo documento

        style type="text/css">
            ...
        /style>

En un archivo externo, dentro del header:

        link rel="stylesheet" type="text/css" href="/css/estilos.css" media="screen" />

Dentro de un elemento HTML:

        p style="color: black; font-family: Verdana;">Un párrafo de texto.

Selectores de Clase

La idea es definir un estilo con un nombre, empezando con “.”:

        .titcol{
            font-size : 10pt;
            font-family : verdana,arial,helvetica;
            font-weight : bold;
            background-color : #F3F1D1;
            background-image : url(images/fontit2.gif);
            color : #333333;
            border-bottom: 1px solid #cccccc;
        }

Y luego:

        td class="titcol">Comentarios sin revisar

Selectores de ID

Es una manera de aplicar estilos a
elementos únicos sin “tocar” el html. 

        #titcol{
            font-size : 10pt;
            font-family : verdana,arial,helvetica;
            font-weight : bold;
            background-color : #F3F1D1;
            background-image : url(images/fontit2.gif);
            color : #333333;
            border-bottom: 1px solid #cccccc;
        }

Y luego:

        td id="titcol">Comentarios sin revisar
