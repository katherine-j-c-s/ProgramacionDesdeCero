Formato de texto

El lenguaje HTML permite dar varios
tipos de formatos al texto de nuestras
páginas. Para ello usa una serie de
etiquetas:

  • Etiquetas de estilo lógico
  • Etiquetas de estilo físico

Etiquetas de estilo físico

Especifican exactamente cómo debe ser formateado el texto

  • Texto en negrita : se usa la etiqueta /b>
  • Texto en cursiva : se usa la etiqueta /i>
  • Tachar una palabra o frase : se usa la etiqueta /strike>
  • Aumentar el tamaño del texto : se usa la etiqueta /big>
  • Reducir el tamaño del texto : se usa la etiqueta /small>
  • Poner subíndices y superíndices : se usan las etiquetas /sub> y /sup>
  • Texto subrayado : se usa la etiqueta /u>

Etiquetas de estilo lógico

Estas etiquetas simplemente describirán el
texto, y el navegador lo mostrará de la manera
más conveniente

  • /strong>: sirve para indicar que el texto
    que modifique es de mayor importancia que el
    resto. Los navegadores gráficos en general
    muestran el texto en negrita.
  • T/em>: de EMphasize, suele mostrar el
    texto en cursiva.
  • /em>: se usa la etiqueta /strike>
  • /code>: texto que forma parte de un
    código. Será mostrado con un tipo de letra
    #monoespaciada "ejemplo.h"
  • /var>: delimita texto escrito por el
    usuario, variables, argumentos de
    comandos...Se muestra como en cursiva .
  • /cite>: Esta etiqueta suele ser usada
    para introducir una cita o una referencia.
    También se muestra en cursiva .
  • /dfn>: usada para introducir una
    definición. Igualmente se muestra en cursiva .
