Listas en HTM

Se definen los siguientes tipos de listas:

  • Numerada u ordenada

    La lista queda delimitado por la etiqueta /OL> y Cada elemento con la etiqueta /li>

    1. elemento 1
    2. elemento 2
    3. elemento 3

  • No ordenada

    La lista queda delimitado por la etiqueta /UL> y Cada elemento con la etiqueta /li>

    • elemento 1
    • elemento 2
    • elemento 3

  • Listas de glosario o de definición

    La lista queda delimitado por la etiqueta /DL> y Cada elemento con la etiqueta /li>

    La inserción de elemento está compuesta de dos partes:

  • La palabra que queremos definir: usaremos la etiqueta /DT>
  • La definición o descripción de esa palabra: usamos la etiqueta: /DD>

Enlaces hipertexto

Para crear enlaces se usa la etiqueta /a>

Ejemplo

tambien puede rediriguir a otra parte de la pagina que estamos, como un ancla

  1. Creando anclas:

    a name="nombreAncla">Comienzo /a

  2. Este texto no tiene ningún significado
    especial y será interpretado por los
    navegadores de igual forma que el texto
    de alrededor

  3. Luego se crea un enlace hacia el ancla

    a href="#nombreAncla ">Aquí /a

  4. El secreto está en el “#”
